# mzProject2
mukza project version2, and based on Vue + ExpressJS

## 개발환경
- OS: Window, Mac
- Front-End: Vue
- Back-End: nodeJS(expressJS)
- Database: mariaDB

### 설치과정
- 원격 리포지토리 복사: git clone https://github.com/Hulkong/mzProject2.git
- 실행: cd backend && npm start
